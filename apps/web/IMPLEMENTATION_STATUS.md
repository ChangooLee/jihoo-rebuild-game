# Jihoo Quest 구현 현황

## 완료된 기능 (Core Infrastructure)

### 1. 프로젝트 기반 구조 ✅
- Next.js 14+ (App Router) + TypeScript
- PWA 설정 (next-pwa)
- Tailwind CSS + shadcn/ui
- Dexie.js IndexedDB 스키마

### 2. 핵심 엔진 모듈 ✅
- **FSRS 엔진**: 분산학습 스케줄링
- **적응 난이도**: 1-up-1-down 알고리즘 + 속도 보정
- **개인화 스케줄러**: 과목 분포 조정, 약점 태그 기반 출제

### 3. 세션 시스템 ✅
- 세션 플로우 관리 (워밍업 → 라운드 A/B/C → 휴식 → 리콜 보스 → 리포트)
- 포커스 타이머 (120/150/180초 프리셋)
- 강제 휴식 시스템 (박스 호흡 4-4-4-4 애니메이션)

### 4. 게임 구현 ✅
- **수학**: 스피드 연산
- **영어**: 듣기 (TTS en-GB), 말하기 (STT)
- **과학**: 원인-결과 연결
- **사회**: 시나리오 선택형
- **워밍업**: Stroop 테스트

### 5. 콘텐츠 시스템 ✅
- Zod 스키마 검증
- YAML/JSON 로더

### 6. 리콜 시스템 ✅
- 리콜 보스 (FSRS Due + 오답 + 약점 태그)
- 오답 변주 시스템

### 7. 데이터 관리 ✅
- JSON 내보내기/가져오기
- 데이터 삭제 기능

### 8. 분석 지표 ✅
- KPI 수집 모듈 (세션 완료율, 평균 세션 길이, 재정복률, 휴식 스킵률)

### 9. 기타 ✅
- 집중모드
- 오디오 모듈 (TTS/STT)

## 핵심 아키텍처

프로젝트는 모듈화된 구조로 설계되어 있으며, 각 모듈은 독립적으로 테스트 및 확장 가능합니다:

```
apps/web/
├── modules/          # 핵심 로직 모듈
├── game/            # 게임 구현
├── components/      # UI 컴포넌트
├── lib/             # 유틸리티 및 타입
└── app/             # Next.js 페이지
```
